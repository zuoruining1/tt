Hexo博客配置代码高亮

Sep 10 2017

博客根目录安装

npm i -S hexo-prism-plugin --save

编辑/_config.yml(默认true改成false)：

highlight：
   enable：false

编辑/_config.yml添加以下内容：

prism_plugin:
  mode: 'preprocess'    # realtime/preprocess
  theme: 'default'
  line_number: false    # default false
